This file illustrates the use of bilan

Let us consider computers in clouds, for which we perform maintenance.
We have 4 clouds, A, B, C and D.
In each cloud, we have computers.
The name of a computer is Xn, where X is the cloud and n a number.
Activities is a maintenance act (e.g. reboot, reinstall, repair, debug ....)

Read the following files.

- data.xlsx Contains the data (from A1). First line is a header of codes.
- codes.txt  Associate a readable definition from a tag.
- groups.txt Describe groups of codes
- document.py This is a python file describing the document.

Run document.py and get the report.
